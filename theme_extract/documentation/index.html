LUXE HOTEL - Luxury Hotel Booking Website Template

Introduction

Thank you for purchasing the LUXE HTML5 template! This documentation will guide you through the installation and customization process to help you get the most out of your new website. LUXE is a modern HTML template designed to load quickly and it is ideal for booking or presenting hotels, motels, spa centers, restaurant or other similar retail businesses. It is fully responsive, optimized with mobile-first techniques, scalable font icons and retina display support. Easy to customize and adapt to your specific needs, this template will impress your audience at first sight.

Installation

  1. Unzip the downloaded file and locate the "html" folder.
  2. Open the "index.html" and "index-2.html" files in your preferred web browser to view the template.
  3. To make changes to the template, open the template files in your preferred code editor (e.g., Sublime Text, Atom, Visual Studio Code).

Customization

The LUXE HTML5 template is fully customizable, allowing you to tailor the design to your needs. Here are some tips to help you get started:

1. Install Sass:

1. Blank is built with sass you will need to install SASS on your machine follow instruction from this link:

https://sass-lang.com/install

2. Run Watch Command to compile:

After the successful installation you will need to run following command using CMD prompt on main containing folder 'html':

sass --watch assets/sass:assets/css

After these two steps now you are good to go for style changing!
